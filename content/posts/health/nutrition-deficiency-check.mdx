# 栄養不足チェック診断｜症状からわかる欠乏栄養素

**「バランスの良い食事を心がけています」**——そう答える人の約7割が、実は何らかの栄養素が不足しているという調査結果をご存知でしょうか。

厚生労働省の国民健康・栄養調査によると、現代の日本人は鉄、亜鉛、ビタミンD、マグネシウムの摂取量が推奨量を下回っている人が多いとされています。見た目には健康でも、体の中では「隠れ栄養不足」が静かに進行している可能性があるのです。

**結論：あなたの不調は、特定の栄養素の欠乏から来ている可能性が高いです。**

ただし、どの栄養素が足りないかは、あなたの生活習慣や食事パターン、そして現れている症状によって異なります。今日は症状から逆算して、あなたに不足している栄養素を特定していきましょう。

---

## なぜ「普通の食事」で栄養不足になるのか

「毎日ちゃんと食べているのに、なぜ？」

そう思うのは当然のこと。しかし現代の食環境には、栄養不足を招く構造的な問題が潜んでいます。

### 土壌の栄養価低下

研究によると、過去50年で野菜に含まれるミネラル量は平均20〜30％減少したと報告されています。大量生産を優先した農法により、土壌そのものが痩せてしまった結果です。

### 加工食品の増加

コンビニ弁当、レトルト食品、冷凍食品。便利さと引き換えに、加工過程で失われるビタミンB群やミネラルは少なくありません。

### ストレスによる消費増加

驚くべきことに、ストレスを感じるだけでビタミンCやマグネシウムの消費量は跳ね上がります。仕事で神経をすり減らしている人ほど、栄養の「出ていく量」が多いというわけです。

つまり、**「入ってくる量は減り、出ていく量は増えている」**——これが現代人の栄養状態の実態なのです。

## 症状別・欠乏栄養素チェック表

あなたの症状はどれに当てはまりますか？ 以下の表で、不足している可能性のある栄養素を確認してみてください。

| 主な症状 | 欠乏が疑われる栄養素 | 補給の優先度 |
|---------|-------------------|------------|
| 慢性的な疲労感・だるさ | 鉄分、ビタミンB12 | ★★★ |
| 肌荒れ・ニキビ・乾燥 | ビタミンA、亜鉛、ビタミンB2 | ★★☆ |
| 髪のパサつき・抜け毛 | 亜鉛、ビオチン、タンパク質 | ★★☆ |
| イライラ・気分の落ち込み | マグネシウム、ビタミンD、オメガ3 | ★★★ |
| 風邪をひきやすい | ビタミンC、ビタミンD、亜鉛 | ★★★ |
| 口内炎・口角炎 | ビタミンB2、B6、鉄分 | ★★☆ |
| 足がつる・筋肉のけいれん | マグネシウム、カリウム、カルシウム | ★★☆ |
| 集中力の低下・ぼんやりする | 鉄分、オメガ3、ビタミンB群 | ★★★ |

複数の症状が重なっている場合は、**複合的な栄養不足**の可能性も。特に鉄・亜鉛・ビタミンDは現代人に不足しやすい「三大欠乏ミネラル」として知られています。

## タイプ別に見る栄養不足パターン

あなたのタイプによって最適な改善方法は異なります。自分がどのパターンに近いか、チェックしてみましょう。

### タイプA：仕事漬けの疲労蓄積型

**特徴**：デスクワーク中心、昼食は簡単に済ませがち、睡眠不足

**不足しやすい栄養素**：ビタミンB群、マグネシウム、ビタミンD

ストレスホルモンの分解にビタミンB群が大量消費されます。さらに室内にこもりがちでビタミンD不足にも。「なんとなくだるい」が口癖になっていませんか？

### タイプB：ダイエット志向の制限型

**特徴**：カロリー制限、糖質制限、特定の食品を避けている

**不足しやすい栄養素**：鉄分、亜鉛、タンパク質、ビタミンB12

肉を控えている人は要注意。植物性食品からでは吸収率が大幅に下がる栄養素があります。「痩せたのに肌がボロボロ」——それは栄養不足のサインかもしれません。

### タイプC：胃腸の弱い吸収不良型

**特徴**：胃もたれしやすい、便秘または下痢気味、食が細い

**不足しやすい栄養素**：ビタミンB12、鉄分、マグネシウム、脂溶性ビタミン

食べていても吸収できていないパターン。医学的には「潜在性欠乏症」と呼ばれ、血液検査では見つかりにくいのが厄介なところ。

## 今日から始める栄養改善5つのステップ

では具体的に、どう改善していけばよいのでしょうか。

### 1. 朝食にタンパク質を追加する

卵1個、納豆1パック、ヨーグルト100g——どれでも構いません。朝のタンパク質摂取は、その日の血糖値の安定と栄養吸収効率に大きく影響します。「朝はコーヒーだけ」から卒業するだけで、体感が変わる人は多いものです。

### 2. 「色」で野菜を選ぶ

緑、赤、黄、紫——異なる色の野菜には異なる栄養素が含まれています。1日に3色以上の野菜を食べることを目標に。難しく考えず、「今日の食卓にこの色はあるか？」と確認するだけでOK。

### 3. 吸収を高める食べ合わせを意識する

鉄分はビタミンCと一緒に摂ると吸収率が約3倍に。カルシウムはビタミンDがないと体内で活用されません。データでは、単体で摂るより組み合わせを意識した方が、同じ量でも効果が大きく異なることがわかっています。

### 4. 腸内環境を整える

どれだけ良い栄養を摂っても、腸が荒れていては吸収されません。発酵食品（味噌、キムチ、ヨーグルト）と食物繊維（きのこ、海藻、根菜）を日常的に取り入れましょう。

### 5. 補いきれない部分はサプリメントで補完する

正直なところ、食事だけで必要な栄養素をすべてカバーするのは現実的ではありません。特に鉄、亜鉛、ビタミンDは食事からの摂取が難しい代表格。

ここで重要なのは、**自分に不足している栄養素を把握した上で、それを的確に補えるサプリメントを選ぶこと**です。

## サプリメント選びで失敗しないために

「とりあえずマルチビタミン」という選び方は、実は非効率かもしれません。なぜなら、本当に必要な栄養素の含有量が少なかったり、逆に過剰な成分が入っていたりするケースがあるからです。

効果を実感できるサプリメントを選ぶには、以下の3点を確認してください。

- **含有量が明記されている**（「〇〇配合」ではなく「〇mg」と具体的な数字）
- **吸収率の高い形態を採用している**（鉄ならヘム鉄、マグネシウムならクエン酸マグネシウムなど）
- **第三者機関の品質検査を通過している**

自分の症状と不足栄養素が特定できたら、それに合ったサプリメントを選ぶことで、体感の変化は格段に早くなります。**まずは2〜3ヶ月継続して、体の変化を観察してみてください。**

## よくある質問（FAQ）

### Q1. 血液検査で「正常」と出ても栄養不足の可能性はありますか？

あります。血液検査の基準値は「病気ではない」レベルを示しているだけで、「最適」を意味しません。例えばフェリチン（貯蔵鉄）は基準値内でも30ng/mL以下なら、疲労感や冷えの原因になることが研究で示されています。

### Q2. 複数の栄養素が不足している場合、何から優先すべきですか？

まずは鉄分とビタミンDから始めることをおすすめします。この2つは他の栄養素の代謝にも関わるため、ここが改善されると連鎖的に体調が良くなるケースが多いです。

### Q3. サプリメントを飲むベストなタイミングは？

脂溶性ビタミン（A、D、E、K）は食後に。水溶性ビタミン（B群、C）は食間でもOK。鉄分は空腹時の方が吸収率は高いですが、胃が弱い人は食後でも問題ありません。継続できるタイミングを優先してください。
